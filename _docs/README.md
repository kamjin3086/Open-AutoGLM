## 🌟 核心知识点

### 必须掌握（⭐⭐⭐）

1. **Agent 主循环**: `_execute_step()` 的 9 个阶段
2. **归一化坐标**: 0-1000 相对坐标系统
3. **Think-Answer**: 可解释的决策模式
4. **上下文管理**: 图像清理策略
5. **提示词工程**: 18 条行为规则

### 重要理解（⭐⭐）

1. **多模态消息**: OpenAI 格式的图文混合
2. **动作执行**: 策略模式的应用
3. **ADB 封装**: 设备控制抽象
4. **错误容错**: 多层容错机制
5. **远程调试**: WiFi ADB 连接

### 扩展知识（⭐）

1. **UIAutomator**: UI 元素树解析
2. **OCR 集成**: 文本识别辅助
3. **并发执行**: 多设备管理
4. **学习系统**: 经验回放
5. **模型集成**: 多 VLM 支持
